Sharing The 6ix
Bitmaker Web Dev Final Project

Overview

Sharing the 6ix is an app created in Ruby on Rails for my final project at Bitmaker's Full Stack Immersive Web Development course. I worked with two other web dev students for this assignment, and the goal was to create any type of web app our group wanted.

The Problem

The idea behind the project was to create a goods sharing network where users can lend out or borrow tools, equipment or appliances from other users in their Toronto neighbourhood (or surrounding), reducing the need to buy "one time use" items - and to create a sense of community. Sharing is caring!

The Process

During the early stages of the project planning, we created user stories, user flow mockups, and model structure to flesh out our ideas. The Solution

Once we fleshed out our ideas on the whiteboard, I made an InVision prototype to plan out the UI design. My main role in this project was handling the front end development (styling in SaSS, but we each took turns pair programming while building the database schema, models and controllers. We completed the project within 10 days and presented a demo to an audience of two other Bitmaker cohorts and all our instructors.
While being one of my most challenging projects I've tackled, I learned so much within such a short period of time. I improved my coding, project planning and designing all within less than 2 weeks.
